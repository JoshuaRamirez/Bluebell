# Codex Agent Guide

## Scope
Work in `src/app` and `libs/ui/*`.

## Lint/Test
- Run lint: `pnpm lint`
- Run tests: `pnpm test`

## PR Instructions
- PR title format: `[admin] <summary>`
- Include a summary section with rationale and before/after behavior

## Notes
All code changes must pass lint and type checks.
